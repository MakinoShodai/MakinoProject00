詳解

template<class Derived>
class ACAnyThreadSingleton< Derived >

Abstract singleton class for any thread (CRTP Pattern)

テンプレート引数
DerivedDerived class that inherit from this abstract class themselves

All functions should be thread-safe on the side of classes that inherit from this class. The following must be done in the derived class.

  1. Declare ACSingleton as friend.
  2. Declare destructor in public.
  3. Declare constructor in protected(or private).
