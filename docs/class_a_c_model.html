◆ m_isSubstance

const bool ACModel::m_isSubstance
protected

Is this component for an substance?

Since the layers of the component are applied to all meshes, it was not possible to draw correctly if translucent and opaque textures were mixed in a single model. Therefore, the component for the semi-transparent mesh was created as a variable, and the variables it uses refer to the substance component, making it appear as if it were a single drawing component.
