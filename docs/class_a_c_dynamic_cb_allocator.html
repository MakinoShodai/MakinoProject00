詳解

template<class ConstBuffType>
class ACDynamicCbAllocator< ConstBuffType >

This class allocates data for constant buffer view per object

テンプレート引数
ConstBuffTypetype of constant buffer view

The following must be done in the derived class. Only 'UINT maxUsePerFrame' should be the argument of the constructor of the derived class.
