template<class Derived>
class ACSingletonBase< Derived >

Abstract singleton base class (CRTP Pattern)

テンプレート引数
DerivedDerived class that inherit from this abstract class themselves

This class doesn't have Get function The following must be done in the derived class.

  1. Declare this abstract class as friend.
  2. Declare destructor in public.
  3. Declare constructor in protected(or private).
